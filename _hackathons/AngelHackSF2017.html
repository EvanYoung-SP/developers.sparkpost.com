Challenge

Build Something Awesome using the SparkPost API

Use the SparkPost API to send an email or receive an email and perform an action via inbound domains and relay webhooks.

Judging Criteria

  1. Have a verified Sending Domain configured
  2. or Have an Inbound Domain and Relay Webhook defined
  3. Creative usage
  4. Comprehensive usage

Prize

$1,000 and 1 year of priority support. $7,000 in total value.
