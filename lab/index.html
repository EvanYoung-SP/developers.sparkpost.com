SparkPost Lab

Mini Hacks

Learn how to use SparkPost by completing these three challenges. Complete 2 mini hacks for a chance to win a prize. Complete the third for a second chance to win.
Use the resources provided by the Developer Hub to help you along. Also, feel free to ask our onsite engineers for assistance.

  1. Send an Email Using SparkPost

    Using your new or existing SparkPost account, send an email to developers@sparkpost.com.
    Be sure to do the following:

    • Set the Subject to "SparkPost Lab: Mini Hack #1"
    • In the body of the email, include your first and last name and account email/

  2. Receive data via a Webhook

    Let's demonstrate SparkPost's ability to push to a specified place.
    To complete this challenge, you must complete the following:

    1. Set up a webhook that sends data to RequestBin
    2. Send an email to developers@sparkpost.com
      • Subject: SparkPost Lab: Mini Hack #2
      • Content: whatever you want
    3. View your request bin, copy the output and email it to developers@sparkpost.com
      • Subject: SparkPost Lab: Mini Hack #2 Data
      • Content: output from request bin

  3. Get data via Message Events

    SparkPost also give you the ability to pull specific data about a transmission.
    Do the following to complete this challenge:

    1. Query the API for message events where:
      • The from parameter is the datetime you performed the first mini-hack
      • The recipients parameter contains developers@sparkpost.com
    2. Email the output to developers@sparkpost.com:
      • Subject: SparkPost Lab: Mini Hack #3
      • Content: the message events response
